Web-based Projects

Tic Tac Toe

An unbeatable game of Noughts and Crosses is the next project for Free Code Camp's Front End Development curriculum.

The site allows you to play the game against a artificially intelligent opponent, which uses a JavaScript implementation of the Minimax algorithm to beat the player or force a draw every time.

The player can also choose whether to play first (as X) or second (as O).

Pomodoro Timer

This project is the next part of Free Code Camp's Front End Development curriculum. It is a Pomodoro Clock, which aims to help motivate the user by setting up a timer which will alternate between "Work Time" and "Break Time".

The clock itself is animated using CSS3, and counts down using JavaScript and jQuery. The user can change the durations of both the session and the break, and a tone will play when the timer runs out, in order to alert the user if they do not have the tab in focus at the time.

JavaScript Calculator

The first Advanced project as part of Free Code Camp's Front End Development curriculum. This is simply a calculator created using the functionality of JavaScript.

My application will calculate addition, subtraction, multiplication, division, and modulo operations on two or more numbers, which may include floating-point parts.

The user has the option of either using the mouse to click on the on-screen buttons, or using their keyboard to type in the operations.

Twitch Viewer

This is the final Intermediate level project for Free Code Camp's Front End Development curriculum. It involves using the Twitch's API to check if any of a list of streamers is currently online.

The project uses the JSON format to get the necessary data from the API and then uses javascript to generate the HTML for the user to view.

The user can also filter the output by streamers currently online, those who are offline, or by using a textbox to search by the streamer's display name, or by the content that they are streaming (if they are online).

Wikipedia Viewer

My fifth project for Free Code Camp's Front End Development curriculum.

This website allows the user to search for information, by using Wikipedia's API to query the text they enter. The site then presents a list of results, which will link to the appropriate Wikipedia articles. Alternatively, the user can get a random article from Wikipedia.

Features of this site include a minimalistic design, and more advanced CSS, using animations in conjunction with jQuery to provide a pleasant user experience.

Localised Weather Teller

Using the Open Weather API, this website gets the user's location and feeds back to them some information about the weather outside.

Features include a background image which changes based on both the time of day and the weather, and the ability to switch between metric and imperial units. The UI is also responsive to the size of the screen (in theory).

This was another project for Free Code Camp's Front End Development curriculum.

Schleicher's Sheep

The first website I have written for Free Code Camp's Front End Development curriculum, as part of the Build a Tribute Page project.

I decided to not be too serious about it, and so injected the project with a little of my interest in Historical Linguistics and Constructed Languages.

Schleicher's Fable is a short parable written in the reconstructed Proto-Indo European language. My tribute page is to the sheep who features in the story, and includes a translation in the fictional language, Esêniqh.

Academic Papers

The Role of Metaphor as an Instrument of Lexical Unit Construction in French

This project was completed as part of my Erasmus year abroad, studying in France. The goal was simply to write on some aspect of the French language, so I chose to write on language evolution, due to my interest in Historical Linguistics.

The paper discusses the movement of meaning of a word from concrete to abstract, eventually causing new words to be coined, and how metaphor plays a part in this.

Constructed Languages

Thenaeös

This is a language which was an attempt to create a naturalistic language based on Proto-Indo European. Thenaeös has a vaguely Romance feel to it, with elements of Nordic.

It was to evolve via realistic stages into a language that could be used by the natives of the fiction world I was slowly creating.

At time of writing, the development of this language is on hiatus.

Esêniqh

A departure from my typical language constructions, Esêniqh was created to sound more guttural than the Romance-based languages I usually work on.

It features a grammar that borrows from Irish in some respects, but is overall original to me, with ergative-absolutive alignment, fusional morphology, head-final clauses, and some unlikely phonetic rules.

It has recently been revived and is currently under reconstruction.

Galelta

Galelta is an experiment in cooperative language construction. Though it has not seen much attention recently, it may yet resurface.

Features include a mostly isolating morphology and subject nouns carrying the temporal information of a clause, rather than the main verb.
